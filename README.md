# VaultOps: Secure Secrets & App Onboarding Platform

This project demonstrates a secure secrets management workflow using HashiCorp Vault, including:

- Vault setup using Docker
- Secure storage of secrets (API keys, DB credentials)
- Policy-based access control
- Application onboarding simulation
- Integration with CI/CD concepts
- Terraform automation (coming next)

## Tech Stack
- HashiCorp Vault
- Docker
- Terraform (WIP)

## Goal
To simulate how applications securely retrieve secrets using Vault in a DevOps environment aligned with PAM/IAM best practices.
